A command-line expense tracking app built with Python and Pandas.

## Features
- Add and save expenses to CSV
- View all expenses with total
- Summarize spending by date or month
- Visualize spending with bar, pie and line charts
- Delete all expenses

## Tech Stack
- Python
- Pandas
- Matplotlib

## How to Run
1. Install dependencies: pip install pandas matplotlib
2. Run: python expense_tracker.py
